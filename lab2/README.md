# 第二次实验题目如下：

1.输出每个月的天数：输入年份，输出该年1~12月每个月的天数。判断闰年的条件是：能被4整除但不能被100整除，或者能被400整除。

2.求最小值，通过键盘输入任意3个整数，输出其中最小的数。

3.在计算机内以1个字节（8个二进制位）来表示char类型。输入1个字符，按照如下格式输出代表该字符字节每一位的值(1个字节中高位为eighth bit, 低位为 first bit):

The eighth bit is  x,
The seventh bit is x,
… …
The first bit is x.

4.求1到n的和：输入一个正整数n,计算序列1+2+3+…+n之和。

5.编写一个程序，输入一个正整数n，计算并输出从1到n之间所有奇数的和。

6.设计一段程序，接收的输入为1个正数，判断它是否为素数并输出结果 (yes或者 no）.

7.输入1个比较大的正数x，找到[1, x]范围内的所有素数并打印出来。

8.编写一个程序，输入一个整数，将其反转并输出。例如，输入12345，输出54321。

9.打印菱形星号“*”图案。输入一个正整数n（n为奇数），打印一个高度为n的“*”菱形图案。例如，当n为7时，打印出如下图案。试编写相应程序。

    *
   ***
  *****
 *******
  *****
   ***
    *

## 📂 文件目录与功能说明
