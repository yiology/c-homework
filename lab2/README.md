
## 文件目录与功能说明

| 文件名 | 功能描述 |
| :--- | :--- |
| `task1.c` | 输出指定年份 1~12 月的天数 | 
| `task2.c` | 输入三个整数，输出最小值 | 
| `task3.c` | 解析并输出字符的 8 位二进制位 |
| `task4.c` | 计算 1 到 n 的累加和 | 
| `task5.c` | 计算 1 到 n 之间所有奇数的和 |
| `task6&7.c` | 素数判断及 [1, x] 范围素数筛选 |
| `task8.c` | 整数反转输出 (如 123 -> 321) |
| `task9.c` | 打印高度为 n 的菱形图案 | 

---

##  如何编译与运行

本项目所有代码均在 Linux 环境下使用 `gcc` 编译器测试通过。

### 编译示例
以“打印菱形”为例，在终端输入：
```bash
