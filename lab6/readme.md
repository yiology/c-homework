## 功能特性

1. **核心引擎**：15x15 标准棋盘，支持精准的行列坐标落子。
2. **胜负判定**：实现了一个高效的 8 方向扫描算法，能够实时判定横、竖、斜四个维度是否存在五子连珠。
3. **AI 智能体**：
   - 算法：基于**权值评分 (Heuristic Evaluation)** 的贪心策略。
   - 策略：AI 会同时评估当前棋盘的“进攻分”（自身连子潜力）与“防守分”（玩家连子威胁），自动选择评分最高的坐标。
